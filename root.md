## Features of this project

- **Local LLM Model** probably using Ollama that will run on any device when installed to fulfil the 6 areas of Polaris that will be expanded upon. **Note this is aimed for apps that can be downloaded**.

- **Local RAG Study Agents** parse a vector DB to get accurate data to use for studying either offline or online, increases accuracy in studying but takes up more storage space on devices. **Note this is aimed for apps that can be downloaded**.

- **Fitness MD Agents** look at uploaded files and data types to analyze a user's fitness progress and then create a growth plan for them to succeed in their physical growth.

## 6 Areas

- **Flashcard Creation**
- **Quizzing**
- **CV Builder**
- **Advisor**
- **Citation Generator**
- **Essay Helper**
